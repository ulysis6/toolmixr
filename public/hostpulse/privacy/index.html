HostPulse Privacy Policy

Last updated: 2026-06-12

HostPulse is a Chrome extension that helps Airbnb hosts track and compare listings. All data stays in your browser — no data is collected, uploaded, or shared.

Data Storage

  • LOCAL ONLY Browsing history (Airbnb listing URLs and public listing data)
  • LOCAL ONLY Your marked listings and favorites
  • LOCAL ONLY Language preference

All data is stored exclusively in chrome.storage.local on your device. You can clear all data at any time from the extension Settings tab.

Data NOT Collected

  • NOT COLLECTED Personal information (name, email, account data)
  • NOT COLLECTED Login credentials or cookies
  • NOT COLLECTED Payment information
  • NOT COLLECTED Browsing history outside Airbnb listing pages
  • NOT COLLECTED Private messages or communications
  • NOT COLLECTED Geolocation data

Permissions

storage — Used solely to save your browsing history and preferences locally. No data is transmitted.

host_permissions (*://*.airbnb.com/*) — Used only to extract publicly visible listing data (price, rating, reviews) from Airbnb listing pages. The extension does not modify any page content or intercept network requests.

Remote Code

HostPulse does not load any remote code. All code is bundled with the extension.

Third-Party Services

HostPulse does not use any third-party analytics, tracking, or advertising services.

Contact

For questions about this privacy policy, please open an issue on the project repository.
