# ToolMixr

Free online web tools for SEO, development, and content creation.

## Tech Stack
- [Astro](https://astro.build/) — Static site framework
- Vanilla JavaScript — All tools run client-side
- Cloudflare Pages — Hosting

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Connected to Cloudflare Pages via GitHub. Push to main branch to deploy.

## Tools
- Meta Tag Generator
- Word Counter
- Robots.txt Generator
- JSON Formatter & Validator
- Base64 Encoder/Decoder
